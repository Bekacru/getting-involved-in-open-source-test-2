Git

Git is a version control system that allows us to track changes, collaborate with other, undo mistakes and manage code history.

Common Git Operations

Initializing

Initialize a new git repository. It'll create a hidden .git folder that'll hold our code history.

git init

Staging

Prepares our changes to be committed. We use "." to specify to stage the whole directory.

git add .

Commit

Commits the changes we made to the git history. And we use `-m [message]` to specify a commit message.

git commit -m [commit message]
